SETUP SUPABASE

Urmează pașii de mai jos — durează ~5 minute:
1. Creează proiect Supabase
Mergi pe supabase.com → Sign up gratuit → New Project
2. Creează tabelul
În proiect → SQL Editor → New Query → Rulează:
create table trail_data (
  id integer primary key,
  data jsonb
);

-- Permite acces fără autentificare
alter table trail_data enable row level security;
create policy "Allow all" on trail_data
  for all using (true) with check (true);
3. Copiază credențialele
Settings → API → copiază Project URL și anon public key
4. Conectează
Lipește URL și key în câmpurile din banner și apasă Conectează
